# Screenshots

This folder contain screenshots of the WorkLog Payment Dashboard for the assessment submission.

1. **dashboard.png** - Main dashboard with stats and recent worklogs
2. **worklogs_list.png** - Worklogs list page with filters
3. **worklog_detail.png** - Single worklog with time entries
4. **payments_filter.png** - Payment page with date range filter
5. **payments_selection.png** - Payment page with worklog selection and exclusions
6. **payment_confirm.png** - Payment confirmation modal
7. **payment_history.png** - Payment history with batch details

## How to Capture

1. Run `docker compose up` to start all services
2. Navigate to http://localhost:5173
